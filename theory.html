Theory of Jacobians

Interactive Learning Module

1. What is a Jacobian?

The Jacobian is a matrix containing the first-order partial derivatives of a set of functions with respect to a set of variables.

It tells us how several functions change simultaneously when the variables change.

2. General Form

For functions f₁, f₂, ..., fₙ of variables x₁, x₂, ..., xₙ:

J = [ ∂fᵢ / ∂xⱼ ]

3. Step-by-Step Procedure

  1. Write the given functions.
  2. Find partial derivatives with respect to each variable.
  3. Arrange them into the Jacobian matrix.
  4. Calculate the determinant of the matrix.
  5. Use the determinant to determine dependence or independence.

4. Example

Given:

f₁ = x + y f₂ = x - y

Partial derivatives:

∂f₁/∂x = 1 ∂f₁/∂y = 1 ∂f₂/∂x = 1 ∂f₂/∂y = -1

Jacobian matrix:

| 1 1 | | 1 -1 |

Determinant:

J = (1)(-1) - (1)(1) = -2

Since J ≠ 0, the functions are independent.

5. Dependency Rule

If J = 0 → Functions are Dependent
If J ≠ 0 → Functions are Independent

6. Dependent Functions Example

f₁ = x + y f₂ = 2x + 2y = 2f₁

Since one function can be expressed in terms of another, the functions are dependent.
