1. Introduction

The Jacobian is a matrix of first-order partial derivatives of several functions with respect to several variables. It tells us how all functions change simultaneously.

J = [ ∂fᵢ / ∂xⱼ ]

2. Dependency Rule

If J = 0 → Functions are Dependent
If J ≠ 0 → Functions are Independent

3. Procedure to Find Jacobian

  1. Write all functions.
  2. Find all partial derivatives.
  3. Arrange derivatives in matrix form.
  4. Compute the determinant.
  5. Interpret the result.

4. Solved Examples

Example 1: Simple Linear 2×2

Given: f₁ = x + y, f₂ = x − y

Jacobian Matrix: | 1 1 | | 1 -1 | J = (1)(-1) - (1)(1) = -2 Conclusion: J ≠ 0, so functions are independent.

Example 2: Polynomial 2×2

Given: f₁ = x² + y², f₂ = xy

∂f₁/∂x = 2x, ∂f₁/∂y = 2y ∂f₂/∂x = y, ∂f₂/∂y = x J = (2x)(x) - (2y)(y) J = 2x² - 2y² Conclusion: Independent except when x² = y².

Example 3: Dependent Functions 2×2

Given: f₁ = x + y, f₂ = 2x + 2y = 2f₁

Jacobian Matrix: | 1 1 | | 2 2 | J = (1)(2) - (1)(2) = 0 Conclusion: Functions are dependent.

Example 4: Trigonometric 2×2

Given: f₁ = sin x + y², f₂ = x eʸ

∂f₁/∂x = cos x, ∂f₁/∂y = 2y ∂f₂/∂x = eʸ, ∂f₂/∂y = x eʸ J = (cos x)(x eʸ) - (2y)(eʸ) J = eʸ(x cos x - 2y) Conclusion: Generally independent.

Example 5: Linear 3×3

Given: f₁ = x + y + z, f₂ = x − y + z, f₃ = x + y − z

Jacobian Matrix: | 1 1 1 | | 1 -1 1 | | 1 1 -1 | J = 4 Conclusion: Functions are independent.

Example 6: Advanced Trigonometric and Exponential 3×3

Given:

f₁ = sin x + y² + eᶻ f₂ = x cos y + z² f₃ = xyz ∂f₁/∂x = cos x, ∂f₁/∂y = 2y, ∂f₁/∂z = eᶻ ∂f₂/∂x = cos y, ∂f₂/∂y = -x sin y, ∂f₂/∂z = 2z ∂f₃/∂x = yz, ∂f₃/∂y = xz, ∂f₃/∂z = xy Jacobian Matrix: | cos x 2y eᶻ | | cos y -x sin y 2z | | yz xz xy | Conclusion: The determinant is generally nonzero, so the functions are independent.

Example 7: Very Difficult Trigonometric Dependency 3×3

Given:

f₁ = sin x + cos y + z f₂ = 2 sin x + 2 cos y + 2z f₃ = x² + y² + z² Observe that: f₂ = 2f₁ Jacobian Matrix: | cos x -sin y 1 | | 2cos x -2sin y 2 | | 2x 2y 2z | Row 2 = 2 × Row 1, therefore two rows are linearly dependent. J = 0 Conclusion: Functions are dependent.

5. Summary

  • Jacobian is the matrix of partial derivatives.
  • Its determinant determines dependence or independence.
  • Zero determinant implies dependency.
  • Nonzero determinant implies independence.
