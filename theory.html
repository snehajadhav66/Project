Theory of Jacobians

Virtual Laboratory on Jacobians

1. What is a Jacobian?

The Jacobian is a matrix of first-order partial derivatives of a set of functions with respect to a set of variables. It describes how a set of functions changes simultaneously with respect to their variables and plays a key role in multivariable calculus and coordinate transformations.

2. General Form

For functions f₁, f₂, ..., fₙ of variables x₁, x₂, ..., xₙ, the Jacobian matrix J is defined as:

J = [∂fᵢ / ∂xⱼ]

    | ∂f₁/∂x₁   ∂f₁/∂x₂   ...   ∂f₁/∂xₙ |
J = | ∂f₂/∂x₁   ∂f₂/∂x₂   ...   ∂f₂/∂xₙ |
    |   ...       ...       ...     ...       |
    | ∂fₙ/∂x₁   ∂fₙ/∂x₂   ...   ∂fₙ/∂xₙ |

3. Steps to Find the Jacobian

  1. Write down the given functions clearly.
  2. Find the partial derivative of each function with respect to every variable.
  3. Arrange these partial derivatives into a square matrix.
  4. Compute the determinant of the Jacobian matrix.
  5. Use the determinant to test functional dependence or independence.

4. Example

Let:

f₁ = x + y
f₂ = x - y

The Jacobian matrix is:

| ∂f₁/∂x   ∂f₁/∂y |   | 1    1 |
| ∂f₂/∂x   ∂f₂/∂y | = | 1   -1 |

Determinant = (1 × -1) - (1 × 1) = -2 ≠ 0

Therefore, the functions are Independent.

5. Dependency Rule

✅ If J = 0 → Functions are DEPENDENT

✅ If J ≠ 0 → Functions are INDEPENDENT

6. Applications

  • Change of variables in multiple integration (e.g. Cartesian to Polar)
  • Coordinate transformations in engineering and physics
  • Solving systems of nonlinear equations (Newton's method)
  • Machine learning — backpropagation and gradient computation
  • Robotics — motion and transformation analysis

7. Summary

  • The Jacobian is a matrix of all first-order partial derivatives.
  • Its determinant indicates functional dependence or independence.
  • It is a fundamental tool in multivariable calculus and engineering mathematics.
