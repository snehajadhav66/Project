Theory of Jacobians

1. What is a Jacobian?

The Jacobian is a matrix of first-order partial derivatives of a set of functions with respect to a set of variables. It is used to study how several functions change simultaneously.

2. General Form

For functions f₁, f₂, ..., fₙ of variables x₁, x₂, ..., xₙ:

J = [∂fᵢ/∂xⱼ]

3. Steps to Find Jacobian

  1. Write the given functions.
  2. Find partial derivatives of each function with respect to every variable.
  3. Arrange these derivatives into a matrix.
  4. Compute the determinant of the matrix.
  5. Use the determinant to test dependence or independence.

4. Example

f₁ = x + y
f₂ = x - y

Jacobian Matrix = 

|  1   1 |
|  1  -1 |

Determinant = -2 ≠ 0, therefore the functions are independent.

5. Dependency Rule

If J = 0 → Functions are Dependent
If J ≠ 0 → Functions are Independent

6. Applications

  • Change of variables in multiple integration
  • Coordinate transformations
  • Engineering analysis
  • Machine learning and optimization

7. Summary

Jacobian = Matrix of partial derivatives.
Determinant of the Jacobian tells whether functions are dependent or independent.
